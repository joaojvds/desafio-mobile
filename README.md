# Desafio para a @Tech
  Projeto de um todo app em React Native + Redux com um consumo de API e dados persistentes.
## Como executar:
  Você deve ter o java e o android studio instalado e configurado para executar no android.
  ```
# Instalar as dependências  
npm install

# Rodar no android
npm run android

# Rodar no ios
npm run ios
```
